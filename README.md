## Usage

Import it:
```css
@font-face {
  font-family:picon;
  src:url(//yne.fr/picon/picon.woff2);
}
```

Declare it:
```css
del { // or .picon, rb, i ...
  font-family:picon;
}
```

Use it :
```html
<del>app</del>
```
to get the app ~~app~~ icon
